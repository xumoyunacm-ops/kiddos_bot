# 🧸 Kiddos — бот подтверждения визитов (сборка для Railway)

Бот спрашивает у клиента **имя → телефон → удобную дату** и присылает
заявку в ваш **групповой чат**.

В этой папке всё, что нужно для запуска на Railway:
- `main.py` — код бота
- `requirements.txt` — зависимости
- `Procfile`, `runtime.txt`, `railway.json` — служебные файлы, чтобы
  Railway понимал, как запускать (раньше их не хватало — отсюда ошибка)

⚠️ Токен и ID группы теперь **не вписываются в код**, а задаются в Railway
в разделе Variables (так безопаснее).

---

## Шаг 1. Создать бота → получить ТОКЕН
1. В Telegram откройте **@BotFather**
2. `/newbot` → задайте имя и username (должен заканчиваться на `bot`)
3. Скопируйте **токен** вида `7712345678:AAH...xyz`

## Шаг 2. Создать группу для заявок
1. Создайте группу, напр. *Kiddos — Заявки*
2. Добавьте в неё вашего бота

## Шаг 3. Узнать ID группы
1. Добавьте в группу **@getmyid_bot**
2. Он покажет `id` вида `-1001234567890` — это и есть ID группы
3. Потом @getmyid_bot можно удалить

---

## Шаг 4. Запуск на Railway

1. Зайдите на **railway.app**, войдите через GitHub
2. **New Project → Deploy from GitHub repo** (или загрузите эту папку
   в репозиторий GitHub и выберите его)
3. Railway сам увидит `requirements.txt` и `railway.json` и соберёт проект
4. Откройте проект → вкладка **Variables** → добавьте ДВЕ переменные:

   | Имя переменной  | Значение                        |
   |-----------------|---------------------------------|
   | `BOT_TOKEN`     | токен из шага 1                 |
   | `GROUP_CHAT_ID` | ID группы из шага 3 (с минусом) |

5. Railway перезапустит проект. Откройте **Deployments → Logs** —
   там должно появиться: `Kiddos bot запущен...`

Готово — бот работает круглосуточно.

---

## Если снова ошибка
Откройте **Deployments → Logs** и пришлите мне последние строки —
там будет видно точную причину. Самые частые:
- забыли добавить `BOT_TOKEN` или `GROUP_CHAT_ID` в Variables →
  бот честно напишет об этом в логах
- `GROUP_CHAT_ID` без минуса → должен быть отрицательным числом

---

## Проверка
1. Откройте вашего бота в Telegram, нажмите **/start**
2. Пройдите 3 вопроса
3. В группе появится заявка:

```
✅ Новое подтверждение визита
👤 Имя: Иван, ООО «Детмаркет»
📞 Телефон: +998 90 123 45 67
🗓 Удобная дата: 25 июня, после 14:00
✈️ Telegram: @ivan
```

## Последний шаг — связать с приглашением
Пришлите мне **username вашего бота** (из шага 1) — я поставлю его
в кнопку «Подтвердить визит», и всё заработает целиком.
